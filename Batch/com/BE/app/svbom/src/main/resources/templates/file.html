API Documentation
List Files
  • Endpoint: GET /file/list
  • Required Parameters:
    • uploadDateStart - Start date for file uploads
    • uploadDateEnd - End date for file uploads
    • templateCode - Template code to filter by
  • Optional Parameters:
    • fileName - Filter by specific file name
    • fileStatus - Filter by file status
  • Response: List of SimoFileMas objects
